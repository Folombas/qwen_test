# Time Web Service

Веб-сервис на Go для отображения текущего времени в реальном времени с современным адаптивным интерфейсом.

## 📋 Описание

Это одностраничное веб-приложение, которое показывает текущее время с обновлением каждую секунду. Приложение включает в себя:

- **Тёмная/светлая тема** с сохранением предпочтений в `localStorage`
- **Интерактивные пузырьки** — анимированные элементы, которые можно «лопнуть» кликом
- **Адаптивный дизайн** — оптимизировано для мобильных устройств, планшетов и десктопов
- **Material Design** — современный визуальный стиль с плавными анимациями

## 🚀 Быстрый старт

### Требования

- Go 1.25.0 или выше

### Запуск

```bash
go run main.go
```

Сервер запустится на порту `8080`. Откройте в браузере:

```
http://localhost:8080
```

### Сборка бинарного файла

```bash
go build -o time-server main.go
```

## 🎨 Функции

### Отображение времени

- Время отображается в формате `ЧЧ-ММ-СС` (часы-минуты-секунды)
- Обновление происходит каждую секунду без перезагрузки страницы
- Дата отображается в формате «сегодня 4 марта 2026 года»

### Переключатель тем

- 🌙 **Тёмная тема** (по умолчанию) — градиентный фон с неоновым свечением
- ☀️ **Светлая тема** — светлый фон с классическим оформлением
- Выбор темы сохраняется в `localStorage` браузера

### Интерактивные пузырьки

- Кнопка «🫧 Активировать пузырьки» в левом верхнем углу
- Пузырьки появляются случайным образом и плавно анимируются
- Клик по пузырьку «лопает» его с анимацией
- Пузырьки исчезают автоматически через 10 секунд

### Адаптивный дизайн

| Устройство | Размер экрана | Размер шрифта времени |
|------------|---------------|----------------------|
| Мобильные | ≤480px | 2.5rem |
| Планшеты | 481-768px | 3.5rem |
| Планшеты большие | 769-1024px | 4rem |
| Десктоп | >1024px | 5rem |

**Дополнительная оптимизация:**

- Ландшафтный режим на мобильных устройствах
- Touch-оптимизация: минимальный размер кнопок 48×48px
- Плавные переходы между темами и состояниями
