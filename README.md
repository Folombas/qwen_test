# 🧠 Go Quiz — Викторина по языку Go

**Веб-приложение для изучения и практики Go с RPG элементами**

---

## 🚀 Особенности

- **118 вопросов** по языку Go
- **RPG-система** с уровнями и опытом
- **12 навыков** в 4 категориях
- **23 достижения** для коллекции
- **Таблица лидеров** с рейтингом
- **Ежедневные квесты**
- **Material Design** UI
- **SQLite** база данных
- **Бэкапы** данных

---

## 🎮 Игровой процесс

### Викторина
Отвечайте на вопросы по Go, получайте опыт (EXP) и повышайте уровень.

### Навыки
Улучшайте 12 навыков в 4 категориях:
- 📚 **GO-НАВЫКИ**: Основы Go, Конкурентность, Интерфейсы, Web, Базы данных, Микросервисы
- 🎯 **ФОКУС**: Мастер Фокуса, Медитация, Борьба с прокрастинацией
- 💪 **СИЛА ВОЛИ**: Сила Воли, Дисциплина
- 💰 **ФИНАНСЫ**: Управление деньгами

### Квесты
Выполняйте 5 ежедневных квестов:
- 30 минут Go
- Борец с искушениями
- Практика кода
- Утренний ритуал
- Цифровой детокс

### Достижения
Коллекционируйте 23 достижения:
- За уровни (5)
- За квесты (3)
- За серию дней (2)
- За искушения (3)
- За боссов (2)
- За изучение Go (2)
- За навыки (2)
- Специальные (4)

## 🎯 Игровая механика

### Формула рейтинга
```
рейтинг = Знание Go × 10 + Фокус × 5 + Сила воли × 3 + Уровень × 100
```

### Очки навыков
```
за уровень = 2 + (уровень / 5)
```

### Характеристики
| Характеристика | Диапазон | Восстановление |
|---------------|----------|----------------|
| Опыт | 0-999999 | Викторина, квесты |
| Уровень | 1-100 | Опыт / 100 |
| Знание Go | 0-100 | Изучение Go |
| Фокус | 0-100 | Отдых, навыки |
| Сила воли | 0-100 | Победы, навыки |
| Дофамин | 0-999 | Достижения, отдых |
